Projects

Projects

Below are Data Science projects on Python, ML, DL, NLP & Generative AI.

Voice Chatbot

Developed a full-stack voice-enabled AI chatbot that enables users to interact with an intelligent system through speech. The application converts voice input into text, processes it using a Large Language Model (Groq API), and returns responses in both text and synthesized speech. The system is built with a modular architecture, using Streamlit for the frontend and a Python-based backend deployed on Render, ensuring scalability and seamless user interaction. This project demonstrates practical implementation of speech processing, API integration, and real-time AI response generation.

Fake News Detection System

An NLP-based Fake News Detection system using TF-IDF feature extraction and Multinomial Naive Bayes classifier to accurately classify news articles as real or fake, achieving 92.5% accuracy. Deployed as a Streamlit web application with a modular ML pipeline, enabling reusable text preprocessing and seamless integration of future models.

Face Mask Detection using CNN

An end-to-end CNN-based Face Mask Detection system for binary image classification (Mask / No Mask), covering data preprocessing, model training, and evaluation with 92% accuracy. Utilized OpenCV for image handling and face region extraction, with data augmentation and hyperparameter tuning for robust performance across varying lighting conditions.

Intelligent Transportation System (ITS)

An AI-powered real-time traffic prediction system to estimate travel time, average speed, and congestion using XGBoost, Linear Regression, LSTM, and GCN models. Integrated real-time TomTom API data with an interactive Streamlit interface, achieving 94.7% accuracy in travel time prediction and an F1-score of 0.91 for congestion detection.
